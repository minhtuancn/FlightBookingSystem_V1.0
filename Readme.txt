FlightBookingSystem
--------------------------------------------------------
A mixed module to search for schedules.

Install:

    1.Create SQL table using the one provided in the folder(realschedulelite.sql)
    2.Upload the files in the order of the folders into the PHPvms software.
    3.Access it by <?php echo url('/FrontSchedules") ;?>
    4.Enjoy.
